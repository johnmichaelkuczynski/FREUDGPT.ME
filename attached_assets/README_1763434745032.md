# Freud Complete Works Database (Part 15)

## Overview

This database contains **83 systematically extracted theoretical passages** from Freud's Complete Works Part 15, totaling **12,728 words** of substantive psychoanalytic content. The database is designed for training AI applications to respond authentically as Freud would, based on his actual theoretical positions rather than generic academic commentary.

## Source Material

**Works Included:**
- The 'Uncanny' (11 passages, 13.3%)
- Beyond The Pleasure Principle (30 passages, 36.1%)
- Group Psychology And The Analysis Of The Ego (42 passages, 50.6%)
- Psycho-Analysis And Telepathy
- Dreams And Telepathy  
- A Note On The Prehistory Of The Technique Of Analysis
- Associations Of A Four-Year-Old Child
- Negation

## Database Structure

### File Formats

1. **freud_database.json** - Complete database in JSON format
2. **freud_database.jsonl** - Line-delimited JSON for training
3. **freud_database.txt** - Human-readable format
4. **freud_database_statistics.json** - Statistics

### Entry Structure

```json
{
  "id": 1,
  "work": "The 'Uncanny'",
  "type": "theoretical",
  "content": "Full passage text...",
  "tags": ["repression", "unconscious", "ego"],
  "word_count": 85
}
```
